<h2 id="sobre">Sobre</h2>
<h3>Este projeto foi desenvolvido com base no que aprendi no curso Desenvolvedor Full Stack no <a href="https://www.devclub.com.br/">DevClub</a> com foco em JavaScript, HTML e CSS visando criar um site que irá sortear uma opção de filme para o usuário assistir dentre as digitadas pelo usuário, esta ideia foi pensada principalmente para auxiliar em momentos de indecisão do usuário ou até mesmo quando temos mais de um usuário com opçãos diferentes e o filme é escolhido pelo sorteador.</h3>
<br>
